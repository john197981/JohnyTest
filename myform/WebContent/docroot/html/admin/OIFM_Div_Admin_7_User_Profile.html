User Profile 

 

Administrators can search, view, and modify some details of user profiles. 

 

a.  Click on "User Profile" button from the left panel of admin main page. 

 

b.  On the "User Profile Search" page, enter various search criteria as needed. Check the checkbox for the corresponding field to display the field details in the search result. Click on "Search" button to do the search based on the entered search criteria 

 

 

 

c.  Click on the NRIC of a user to view the profile of the user 

 

 

 

d.  Make modifications to the role of the user if needed and click on "Save" button to save modifications made.
