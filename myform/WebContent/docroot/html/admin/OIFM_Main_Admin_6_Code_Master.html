Code Master 

 

The Code Master allows administrators to maintain various common codes used throughout the system. The codes covers internal My Forum system codes as well as common codes from PM2S interface. 

 

Click on "Code Master" button from the left panel of admin main page to go to the "Code Master" page 

 

 

 

1.  Search Code 

 

On the "Code Master" page, enter the search criteria. Click on "Search" button to search. 

 

2.  Modify Code Description 

 

a.  Search for the code 

 

b.  From the search result, click on the code to open the code details 

 

c.  Make necessary modifications to the code. Check on the "Obsolete" checkbox to make the code obsolete. Click ok "Save"
